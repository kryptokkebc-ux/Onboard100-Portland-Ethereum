# Onboard 100 Portland Users to Ethereum Self-Custody
Free, open-source curriculum and program to teach 100 Portland-area underserved users (recovery/low-income focus) safe Ethereum self-custody.
## Project Goal
Onboard 100 participants to create and secure their first wallets via 4 interactive Zoom sessions. Focus: MetaMask setup, seed phrases, testnet transactions.
## Target Audience
- Local recovery communities (AA/Zoom groups)
- Low-income/gaming networks (RoK, trading beginners)
- Portland, OR residents new to crypto
## Curriculum Outline
- **Session 1**: Ethereum basics & why self-custody matters
- **Session 2**: Wallet install (MetaMask), seed phrase security
- **Session 3**: First testnet transaction, gas fees
- **Session 4**: Advanced tips, resouces, Q&A
## Planned Milestones & Metrics
| Milestone | Deliverable | Timeline | Success Metric |
|-----------|-------------|----------|----------------|
| 1 | Curriculum docs/slides in repo | Week 2 | Live & licensed |
| 2 | 50 users onboarded | Week 6 | Quiz pass rate 80%+ |
| 3 | 100 total + final report | Week 12 | Retention survey NPS 8+ |
